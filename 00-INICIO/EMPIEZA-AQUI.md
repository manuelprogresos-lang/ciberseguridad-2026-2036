# Empieza aquí

No intentes estudiar todas las carpetas a la vez.

## Tu orden durante el primer mes

1. Linux.
2. Redes.
3. Git.
4. Python.
5. Documentación.

## Primera sesión

Ejecuta desde la raíz del proyecto:

```bash
chmod +x setup.sh
./setup.sh
```

Después abre:

```bash
cat 00-INICIO/DIA-01.md
```

## Método de estudio

Para cada tema:

1. Lee el objetivo.
2. Intenta el ejercicio.
3. Pide pistas a la IA, no la solución completa.
4. Verifica con documentación.
5. Repite sin mirar.
6. Escribe lo aprendido con tus palabras.
7. Guarda un commit en Git.

## Norma de seguridad

No escanees, pruebes ni accedas a redes o sistemas ajenos.
Los ejercicios de este repositorio usan datos sintéticos y tu propia máquina.
