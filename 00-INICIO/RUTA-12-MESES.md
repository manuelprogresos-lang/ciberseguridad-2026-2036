# Ruta de 12 meses

## Meses 1–3
Linux, redes, Git, Python y documentación.

## Meses 4–6
Windows, PowerShell, administración y Active Directory en laboratorio aislado.

## Meses 7–9
HTTP, HTML, JavaScript básico, SQL, APIs, autenticación y desarrollo seguro.

## Meses 10–12
Logs, SIEM, detección, respuesta a incidentes, hardening y proyecto final.

## Resultado esperado

Al terminar el año:
- administrar Linux y Windows a nivel básico;
- explicar redes y HTTP;
- automatizar tareas sencillas;
- analizar logs;
- documentar incidentes ficticios;
- disponer de 4 a 6 proyectos seguros;
- estar preparado para puestos de entrada en soporte, sistemas o SOC.
