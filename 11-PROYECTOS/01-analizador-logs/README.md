# Proyecto 1 — Analizador de logs sintéticos

## Objetivo

Leer un archivo de eventos ficticios y producir un resumen.

## Ejecutar

Desde esta carpeta:

```bash
python3 analizador.py datos.log
```

## Lo que debes aprender

- lectura de archivos;
- separación de campos;
- diccionarios;
- conteo;
- errores;
- salida legible.

## Mejores futuras

- exportar JSON;
- añadir pruebas;
- filtrar por severidad;
- crear una gráfica;
- anonimizar direcciones.
