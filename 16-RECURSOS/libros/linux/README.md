# Libros de Linux

Esta carpeta guarda los libros de apoyo para estudiar Linux dentro de la ruta de ciberseguridad.

## Libros

- `How-Linux-Works.pdf`
  - Mejor para entender como funciona Linux por dentro.
  - Usalo cuando quieras comprender procesos, arranque, discos, red, logs y servicios.

- `linux-basics-for-hackers.pdf`
  - Mejor para practicar comandos y tareas utiles para ciberseguridad.
  - Usalo como laboratorio, siempre en tu propia maquina virtual o entorno local.

## Orden recomendado

1. Primero practica con tus notas de `01-FUNDAMENTOS/01-linux`.
2. Despues lee el tema equivalente en `How-Linux-Works.pdf`.
3. Luego practica ejercicios relacionados en `linux-basics-for-hackers.pdf`.
4. Cierra cada dia escribiendo notas en `laboratorio/notas/`.

## Regla de estudio

No copies comandos sin entenderlos. Antes de ejecutar algo nuevo:

```bash
man COMANDO
COMANDO --help
```

Si el comando modifica, borra, descarga, instala o toca permisos del sistema, primero preguntamos y lo probamos en laboratorio.
