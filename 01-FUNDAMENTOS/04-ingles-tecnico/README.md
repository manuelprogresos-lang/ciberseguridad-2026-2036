# Inglés técnico

## Rutina semanal

- Aprende 10 palabras.
- Lee una página de documentación.
- Resume un concepto en español.
- Escríbelo después en inglés sencillo.
- Pronuncia los términos en voz alta.

## Formato de vocabulario

| English | Español | Ejemplo técnico |
|---|---|---|
| permission | permiso | File permission |
| process | proceso | Running process |
| log | registro | System log |
