## Libros de apoyo

```text

```

Orden recomendado:

1. Seguir primero las notas por dia de esta carpeta.
2. Leer el tema relacionado en `How-Linux-Works.pdf`.
3. Practicar ejercicios controlados con `linux-basics-for-hackers.pdf`.
4. Guardar dudas y comandos probados en `laboratorio/notas/`.

La ruta detallada esta en:

```text
RUTA-LECTURA-LIBROS.md
```

## Práctica segura

Trabaja en tu propia máquina virtual. No copies comandos destructivos.
Antes de usar un comando, consulta:

```bash
man COMANDO
COMANDO --help
```
