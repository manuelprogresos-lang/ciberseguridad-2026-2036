# Certificaciones opcionales

No compres ninguna certificación durante el primer mes.

Ruta orientativa:
- inicial: fundamentos de redes, Linux y seguridad;
- primer año: certificación general de entrada, si ayuda a tu empleo;
- años 2–3: certificación de tu especialización;
- etapa avanzada: cloud, arquitectura, AppSec o gestión.

Comprueba siempre el temario y requisitos vigentes antes de pagar.
